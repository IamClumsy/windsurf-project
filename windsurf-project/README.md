# Apex Girl Artist Picker

A web application to discover and rate the best artists, built for Apex Girl.

## Prerequisites

- Node.js (v16 or later)
- npm (comes with Node.js)

## Getting Started

1. Install Node.js from https://nodejs.org/
2. Clone this repository
3. Run `npm install` to install dependencies
4. Run `npm run dev` to start the development server
5. Open http://localhost:5173 to view it in your browser

## Features

- Browse a curated list of artists
- View artist details and ratings
- Rate and review artists
- Search and filter artists by genre
- Responsive design that works on all devices

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Icons
